# CNKI Search Strategy for Art Studies

## Keyword construction

For art studies theses, construct search terms in layers:

### Layer 1: Core topic
Extract from thesis title and keywords.
- *Example*: `楚漆器`, `战国楚漆器`, `楚国漆器工艺`

### Layer 2: Thematic expansion
Add synonyms and adjacent art-studies concepts.
- *Example*: `漆器纹样`, `楚文化 漆器`, `战国 漆工艺`, `楚国 装饰艺术`

### Layer 3: Method/theory
Add art-historical or theoretical terms.
- *Example*: `漆器 审美`, `楚漆器 视觉文化`, `漆器 符号学`, `战国 器物 形制`

### Layer 4: Case-specific
Artist names, work titles, archaeological sites, unearthed artifacts.
- *Example*: `曾侯乙 漆器`, `马山一号墓 漆器`, `望山楚墓 漆器`

## Recommended CNKI source categories

For art studies master's theses, prioritize:

| Priority | Source type | DB code | Why |

## Search techniques

1. **Exact phrase**: Use quotes for multi-word terms: `"楚漆器纹样"`
2. **Boolean**: CNKI supports AND (space) and OR: `漆器 纹样 AND 战国`
3. **Date range**: Focus on recent 10-15 years for methodology, include classic works regardless of age
4. **Cited reference search**: Find papers that cite a key paper to trace research lineage
5. **Author tracking**: Note recurring authors and set up alerts
6. **Snowball**: From one relevant paper's references, find 3-5 more papers

## Recording search results

For each search, record:
- Search terms used
- Database and filters
- Date of search
- Total results found
- Number screened
- Number selected for reading
- Why certain papers were excluded

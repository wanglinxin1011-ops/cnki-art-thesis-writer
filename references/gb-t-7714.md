# GB/T 7714-2015 Citation Format Guide

## Journal article [J]
```
[序号] 作者. 题名[J]. 刊名, 出版年, 卷号(期号): 起止页码.
```
```
[1] 张三, 李四, 王五. 战国楚漆器纹样的审美特征研究[J]. 装饰, 2023, 45(3): 88-95.
[2] 赵六. 楚国漆器工艺的社会文化背景[J]. 美术研究, 2022(2): 45-52.
```

## Monograph [M]
```
[序号] 作者. 书名[M]. 出版地: 出版社, 出版年: 引用页码.
```
```
[3] 陈振裕. 楚文化与漆器研究[M]. 北京: 科学出版社, 2003: 228-230.
```

## Dissertation [D]
```
[序号] 作者. 题名[D]. 出版地: 出版单位, 出版年.
```
```
[4] 刘芳. 楚国漆器装饰纹样研究[D]. 武汉: 湖北美术学院, 2020.
```

## Conference paper [C]
```
[序号] 作者. 题名[C]//论文集名. 出版地: 出版者, 出版年: 起止页码.
```

## Online resource [EB/OL]
```
[序号] 作者. 题名[EB/OL]. 出版日期/更新日期 [引用日期]. URL.
```

## Key rules for art studies

1. **Ancient texts**: When citing classical Chinese texts, include the original dynasty and edition:
   ```
   [5] (汉)司马迁. 史记[M]. 北京: 中华书局, 1959: 328.
   ```

2. **Excavation reports**: Use monograph format for archaeological reports:
   ```
   [6] 湖北省博物馆. 曾侯乙墓[M]. 北京: 文物出版社, 1989.
   ```

3. **Art catalogues**: 
   ```
   [7] 中国古代书画鉴定组. 中国绘画全集[M]. 北京: 文物出版社, 1997.
   ```

4. **Multiple authors**: List up to 3 authors; use `等` for 4+:
   ```
   [8] 张三, 李四, 王五, 等. 楚国漆器研究[M]. 武汉: 武汉大学出版社, 2019.
   ```

## In-text citation marks

Use superscript-style bracketed format `[n]` followed by page number:
- `……制作精美的饮食器具——漆杯的景象：[3]228`
- `故一杯棬用百人之力，一屏风就万人之功。[3]247`
- Multiple sources: `……具有重要的历史意义。[2]45-48; [4]15-20`

Do NOT use inline parenthetical formats like `（来源：Author，Title，第X页）`.
